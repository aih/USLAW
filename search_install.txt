We need some additional components:
(All commands must be executed by root user)
apt-get install python-pip
apt-get install solr-common
apt-get install solr-tomcat
pip install django-haystack
apt-get install python-beautifulsoup
pip install pysolr

Next step, configuring Django project.
There are new options we need to add into settings_local.py:

HAYSTACK_SITECONF = 'uslaw.search_sites'  # If project in uslaw directory
HAYSTACK_SEARCH_ENGINE = 'solr'
HAYSTACK_SOLR_URL = 'http://127.0.0.1:8080/solr'

Next step, configuring Solr:
Run in project dir:
  python manage.py build_solr_schema
Drop the XML output in your Solrâ€™s schema.xml (usualy /etc/solr/conf/schema.xml), old schema file need to be deleted.
Restart Solr: 
  service tomcat restart
Next, run:
  python manage.py rebuild_index 

That's All! You can run app.
